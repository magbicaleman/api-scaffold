initAppModules()

This function sets up the global modules for the application. These are accessible as global.services (services such as redis, mongo, etc ../services) global.shared (shared functions ../shared) global.models (mongo models ../services/mongo/models)

initLogging()

This function creates an instance of global.logger. It is used as an alternative to console.log. It has multiple log levels such as info, error, warn, debug and silly. To use it simply call logger.[LOG_LEVEL]('some message');
